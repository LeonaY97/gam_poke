# AI 玩家人设池 (AI Persona Pool)

共 10 名 AI 玩家，覆盖德州扑克 10 种典型风格。分配到同一房间时，尽量避免风格过于相似的 AI 同屋。

---

## 风格分类

| 风格标签 | 类型 | 描述 |
|---------|------|------|
| `tight-aggressive` | 紧凶 (TAG) | 入局率低，但一旦入局就积极加注施压 |
| `loose-aggressive` | 松凶 (LAG) | 入局率高，频繁加注，侵略性极强 |
| `tight-passive` | 紧弱 | 入局率低，入池后以跟注为主，很少加注 |
| `loose-passive` | 松弱 / 鱼 | 入局率高，喜欢跟注看牌，很少弃牌 |
| `maniac` | 疯子 | 超宽范围，几乎每手牌都参与，疯狂加注 |
| `rock` | 岩石 | 极紧，只玩最强起手牌，极难被 bluff |
| `calling-station` | 跟注站 | 几乎从不弃牌，什么牌都跟注到底 |
| `bomber` | 炸弹型 | 平时很紧，拿到强牌就下重注慢打 |
| `tricky` | 狡猾型 | 频繁变换节奏，频繁 bluff，难以捉摸 |
| `short-stack` | 短码型 | 小筹码策略，倾向快速 all-in |

## 人设详情

### 01. 陈志远 — 紧凶教授 (TAG)

| 项目 | 内容 |
|------|------|
| **昵称** | 🧠 志远 |
| **风格标签** | `tight-aggressive` |
| **原型参考** | 经典职业牌手 TAG 风格（类似 Daniel Negreanu 早期） |
| **入局率 (VPIP)** | 18% ~ 22% |
| **翻牌前加注率 (PFR)** | 15% ~ 18% |
| **性格描述** | 前职业扑克选手，退休后在线上娱乐。打法极其标准，位置感极强，翻后决策精准。喜欢用强价值牌慢打，也会在合适的场合做持续下注 (c-bet)。 |
| **策略特点** | - 只玩优质起手牌，位置越靠后范围越宽<br>- 翻牌后击中就打，没击中视情况 c-bet<br>- 面对大加注愿意弃掉中等牌<br>- 很少做 hero call |
| **AI 参数倾向** | `handStrength` 阈值高，bluff 概率低，raise 频率中等 |

### 02. 王浩然 — 松凶野狼 (LAG)

| 项目 | 内容 |
|------|------|
| **昵称** | 🐺 浩然 |
| **风格标签** | `loose-aggressive` |
| **原型参考** | 顶级 LAG 选手（类似 Phil Ivey、Tom Dwan） |
| **入局率 (VPIP)** | 35% ~ 45% |
| **翻牌前加注率 (PFR)** | 25% ~ 30% |
| **性格描述** | 年轻的线上高额桌玩家，打法极具侵略性。擅长在不同牌面构造 bluff，给对手极大压力。喜欢在有利位置频繁骚扰对手。 |
| **策略特点** | - 极宽的加注范围，尤其在按钮位和 cutoff<br>- 翻后擅长读牌，三条街 bluff<br>- 喜欢 float（跟注翻牌等待转牌bluff）<br>- 面对反击也能灵活弃牌 |
| **AI 参数倾向** | `handStrength` 阈值低，bluff 概率高，raise 频率高 |

### 03. 李素芬 — 紧弱阿姨 (Tight Passive)

| 项目 | 内容 |
|------|------|
| **昵称** | 👩 素芬 |
| **风格标签** | `tight-passive` |
| **原型参考** | 线下家庭牌局中的保守型玩家 |
| **入局率 (VPIP)** | 12% ~ 16% |
| **翻牌前加注率 (PFR)** | 5% ~ 8% |
| **性格描述** | 牌局常客，但比较保守。只玩自己看得懂的牌，入池后多半跟注，不太会主动下注。赢了小钱就很开心，输了也不会上头。 |
| **策略特点** | - 只玩大对子和 AK、AQ 这样的强牌<br>- 入池后很少加注，主要是跟注<br>- 翻牌没中就弃牌<br>- 面对大 bet 会很痛苦但还是会弃掉中等牌 |
| **AI 参数倾向** | `handStrength` 阈值高，bluff 概率极低，call 频率高，raise 频率低 |

### 04. 张大宝 — 松弱鱼儿 (Loose Passive)

| 项目 | 内容 |
|------|------|
| **昵称** | 🎰 大宝 |
| **风格标签** | `loose-passive` |
| **原型参考** | 娱乐场典型 "鱼" 玩家 |
| **入局率 (VPIP)** | 50% ~ 65% |
| **翻牌前加注率 (PFR)** | 8% ~ 12% |
| **性格描述** | 富二代，打牌纯粹为了娱乐。什么牌都想看一看翻牌，喜欢追同花和顺子。不太会弃牌，总觉得"万一中了呢"。是牌桌上最好的顾客。 |
| **策略特点** | - 超宽的入池范围，什么牌都 limp（跛入）<br>- 很少主动加注，基本都是跟注<br>- 喜欢追同花、顺子，哪怕赔率不够<br>- 翻牌有对就会跟注到河牌 |
| **AI 参数倾向** | `handStrength` 阈值极低，bluff 概率低，call 频率极高，fold 频率低 |

### 05. 赵猛龙 — 疯子玩家 (Maniac)

| 项目 | 内容 |
|------|------|
| **昵称** | 🔥 猛龙 |
| **风格标签** | `maniac` |
| **原型参考** | 线上低额桌常见的疯狂玩家 |
| **入局率 (VPIP)** | 70% ~ 90% |
| **翻牌前加注率 (PFR)** | 40% ~ 60% |
| **性格描述** | 脾气火爆的老板，打牌就是为了爽。几乎每手牌都加注，不是在加注就是在 all-in。短筹码更是疯狂推推乐。运气好的时候能赢一大堆，运气差的时候很快就上桌。 |
| **策略特点** | - 几乎所有牌都加注入池<br>- 翻牌后不管中没中都继续打<br>- 经常 all-in 抢盲注<br>- 完全不考虑对手范围，只看自己牌 |
| **AI 参数倾向** | `handStrength` 阈值极低，bluff 概率极高，raise 和 all-in 频率极高 |

### 06. 孙守义 — 岩石玩家 (Rock)

| 项目 | 内容 |
|------|------|
| **昵称** | 🪨 守义 |
| **风格标签** | `rock` |
| **原型参考** | 线下老牌局中极紧的老玩家 |
| **入局率 (VPIP)** | 8% ~ 12% |
| **翻牌前加注率 (PFR)** | 6% ~ 10% |
| **性格描述** | 退休老会计，打牌极其严谨。只玩最强的起手牌，等待时间极长。一旦入局基本就是大牌。很难被 bluff，因为他入局的牌你很难赢。 |
| **策略特点** | - 只玩 JJ+、AK 这种顶级起手牌<br>- 入池必加注或 3-bet<br>- 翻后基本不会弃掉顶对<br>- 极难被 bluff，只信自己的牌 |
| **AI 参数倾向** | `handStrength` 阈值极高，bluff 概率极低，fold 频率高（未入局时），入池后 call 频率高 |

### 07. 周彩莲 — 跟注站 (Calling Station)

| 项目 | 内容 |
|------|------|
| **昵称** | 🌸 彩莲 |
| **风格标签** | `calling-station` |
| **原型参考** | 娱乐型女玩家，什么都不舍得弃 |
| **入局率 (VPIP)** | 45% ~ 55% |
| **翻牌前加注率 (PFR)** | 3% ~ 6% |
| **性格描述** | 牌局中的常客，性格温和，打牌全凭感觉。几乎从不主动加注，也几乎从不弃牌。你下注她就 call，你 all-in 她也想 call。是 bluff 的噩梦。 |
| **策略特点** | - 很宽的入池范围，几乎都是 limp 或 call<br>- 翻牌有任何对子、听牌就跟注到底<br>- 河牌即使只有高牌也可能 hero call<br>- 完全不会 bet 或 raise |
| **AI 参数倾向** | `handStrength` 阈值极低，call 频率极高，fold 频率极低，几乎不 raise |

### 08. 吴大炮 — 炸弹型 (Bomber)

| 项目 | 内容 |
|------|------|
| **昵称** | 💣 大炮 |
| **风格标签** | `bomber` |
| **原型参考** | 线下牌局中平时沉默、拿到nuts就下重注的玩家 |
| **入局率 (VPIP)** | 15% ~ 20% |
| **翻牌前加注率 (PFR)** | 10% ~ 15% |
| **性格描述** | 话不多的中年男人，平时打得很紧很稳。但一旦拿到强牌，就会下很重的注，甚至慢打引你上钩。赢的时候总是赢大锅，输的时候都是小锅。 |
| **策略特点** | - 平时很紧，只玩强牌<br>- 拿到坚果牌时慢打（慢玩）引对手下注<br>- 强牌下注尺度很大（超池下注）<br>- 很少 bluff，基本都是价值下注 |
| **AI 参数倾向** | `handStrength` 阈值高，bluff 概率低，强牌时 bet/raise 金额大 |

### 10. 陈铁头 — 短码推土机 (Short Stack)

| 项目 | 内容 |
|------|------|
| **昵称** | 🪖 铁头 |
| **风格标签** | `short-stack` |
| **原型参考** | 锦标赛后期短码策略 / 现金桌买入短码推推乐玩家 |
| **入局率 (VPIP)** | 20% ~ 30% |
| **翻牌前加注率 (PFR)** | 18% ~ 28% |
| **性格描述** | 专门玩短码的玩家，信奉"短码就是要玩得凶"。要么不玩，要玩就 all-in。精通 push/fold 策略，对各位置 shove 范围了如指掌。 |
| **策略特点** | - 筹码少时倾向直接 all-in<br>- 对 all-in 范围掌握精准<br>- 翻牌后基本没有决策，要么 all-in 要么弃<br>- 很擅长抢盲注 |
| **AI 参数倾向** | all-in 频率高，筹码越少越激进，fold 牌时果断弃 |

## 风格相似度矩阵

分配 AI 时尽量避免同屋中相似度 > 0.7 的组合。

| 风格对 | 相似度 | 说明 |
|--------|--------|------|
| maniac ↔ loose-aggressive | 0.75 | 都很激进、范围宽，区别只在程度 |
| loose-passive ↔ calling-station | 0.85 | 都是跟注型玩家，极难被 bluff |
| tight-passive ↔ rock | 0.80 | 都很紧，区别只在紧的程度 |
| bomber ↔ rock | 0.65 | 都偏紧，但 bomber 强牌下重注，rock 偏被动 |
| tricky ↔ loose-aggressive | 0.70 | 都有侵略性，但 tricky 更重策略变化 |
| short-stack ↔ maniac | 0.60 | 都很凶，但 short-stack 有策略，maniac 纯疯 |
| tight-aggressive ↔ bomber | 0.55 | 都偏紧凶，区别在于 bomber 更极端 |
| tight-aggressive ↔ tricky | 0.50 | 都有技术含量，但 tricky 更花 |

## 分配策略

1. **随机抽取**：从 10 人池中随机选择需要的数量
2. **多样性过滤**：如果选中的 AI 中有多对相似度 > 0.7 的组合，则重新抽取
3. **最多重试 5 次**：避免无限循环，5 次后取当前最优解
4. **不同房间独立分配**：每个房间的 AI 组合互不影响
